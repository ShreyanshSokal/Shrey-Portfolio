# Shrey-Portfolio

Shrey's Portfolio is a responsive personal website designed to showcase the skills, education, projects, and professional experiences of Shreyansh Sokal. The website is built using HTML, CSS, and JavaScript, with Bootstrap for styling and layout.

Features

Home Page: Introduction and personal information about Shreyansh Sokal.
Education Section: Detailed education history with institutions, durations, and specializations.
Skills Section: A list of technical skills with logos and proficiency levels.
Internships Section: Information on internships with details on roles and technologies used.
Projects Section: Showcase of interesting projects with brief descriptions and images.
Contact Section: Contact information and links to social media profiles.
Responsive Design: Ensures an optimal viewing experience across various devices (desktops, tablets, and mobile phones).
Dark Mode Toggle: A button to switch between light and dark themes.

Technologies Used
HTML5: Markup language used for structuring the website content.
CSS3: Styling language used for the presentation of the website.
JavaScript: Used for functionality like dark mode toggle and smooth scrolling.
Bootstrap: CSS framework for responsive design and layout.
FontAwesome: Icon library used for social media and contact icons.
